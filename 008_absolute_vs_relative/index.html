Path to use in 'src' attributes

The name of the file alone means the file is in the same folder as the file calling it

A path of folder names and then a file name means the file path can start from the same folder the code file is in. There's no root "/" indicator in the front.

In src="../img/buildings/capitol.jpg" the ".." means go up a folder level and then dig down.

Absoulte urls have complete url paths or paths from the root folder in your system.
